How Our SVM Model Works

Step 1: Face Detection

Using OpenCV's Haar Cascade classifier, we first detect and isolate the face in the image:

face_cascade = cv2.CascadeClassifier('haarcascade_frontalface_default.xml')
faces = face_cascade.detectMultiScale(gray_image, 1.3, 5)

Step 2: Facial Landmark Detection

We identify 68 key facial landmarks using dlib's predictor:

predictor = dlib.shape_predictor("shape_predictor_68_face_landmarks.dat")
landmarks = predictor(image, face_rect)

Step 3: Feature Extraction

Calculate distances and angles between landmarks to create feature vector:

features = []
for i in range(68):
  for j in range(i+1, 68):
    distance = euclidean_distance(landmarks[i], landmarks[j])
    features.append(distance)

Step 4: SVM Classification

Our trained SVM model classifies the emotion based on the extracted features:

svm_model = SVC(kernel='rbf', C=1.0)
prediction = svm_model.predict([features])
emotion = emotions[prediction[0]]
